# Emprest-Cartel Landing Page

Landing page Next.js para simulação de empréstimo com integração Meta Ads Pixel.

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse `http://localhost:3000`

## Build

```bash
npm run build
npm start
```
